This program implements a binary search tree (BST) in Java. It reads a text file that contains a series of commands and strings. The available commands are:


1. insert [string]: insert a string into the BST
2. search [string]: search for a string in the BST
3. print: print the contents of the BST in inorder traversal
4. largest: print the largest string in the BST
5. smallest: print the smallest string in the BST
6. height: print the height of the BST
7. size: print the number of strings currently stored in the BST




To use this program, run it with the file as a command line argument like this: java Lab4.java commands
